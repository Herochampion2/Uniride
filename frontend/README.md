# UniRide - Dynamic Carpool System for University Students

This project is a dynamic carpool system designed to help university students commute more efficiently. The system will connect students with similar routes, allowing them to share rides on a regular basis.

## Features

*   **User Profiles:** Students can create profiles with their university, schedule, and regular commute details.
*   **Route Matching:** The system will intelligently match students with similar commuting routes.
*   **Dynamic Ride Requests:** Students can request and offer rides for specific dates and times.
*   **Real-time Notifications:** Users will receive real-time notifications about ride requests, confirmations, and updates.
*   **Rating and Review System:** A feedback system to build trust and accountability within the community.
